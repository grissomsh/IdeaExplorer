Application Life cycle - Add Accessibility
This idea will ensure that EMC products are in alignment with industry standards around Accessibility requirements and will improve the overall customer experience especially in the federal space.
This was discussed as part of the brainstorming exercise done by PSRC Team.
The solution is to have a common accessibility framework that defines, how to design and test for accessibility with the right set of tools, this should be self-service and guided. The output of the testing should be plugged into a tool which then outputs a VPAT and a accessibility rating which then should be fed into the Application Life cycle. This tool should store all versions and their rating.
